{% blocktrans %} Le site Dezède implique la mise en œuvre de plusieurs traitements de données à caractère personnel décrits dans le présent document. {% endblocktrans %}

{% url 'presentation' as presentation_url %} {% blocktrans %} Ces traitements sont mis en œuvre sous la responsabilité des propriétaires du site. Voir la section Présentation. {% endblocktrans %}

{% blocktrans %} Pour toute question sur vos données, les propriétaires du site peuvent être contactés à l’adresse dezede.contact@gmail.com. {% endblocktrans %}

{% blocktrans %} L’université de Rouen Normandie assure la gestion du site internet. Ainsi pour toute question sur la protection de vos données, vous pouvez également contacter la déléguée à la protection des données de l’établissement à l’adresse dpo@univ-rouen.fr. {% endblocktrans %}

{% blocktrans %} Si vous estimez, après nous avoir contactés, que vos droits et libertés ne sont pas respectés, vous pouvez également introduire une réclamation auprès de la CNIL. {% endblocktrans %}

{% trans 'Finalité et base légale du traitement' %}

{% blocktrans %} Afin de contribuer au site, il est nécessaire d’obtenir la création d’un compte contributeur. Il permet au contributeur d’alimenter la base de données et assure la traçabilité des publications. {% endblocktrans %}

{% blocktrans %}

Afin de gérer ce compte, les données suivantes relatives au contributeur sont traitées :

  • son identité
  • son adresse mail
  • sa biographie (facultative)
  • ses publications
  • son mot de passe (chiffré)
{% endblocktrans %}

{% blocktrans %} Le traitement de ces données repose sur les missions d’intérêt public (recherche et formation) dont sont investis les propriétaires du site (universités et laboratoires). {% endblocktrans %}

{% trans 'Durée de conservation et destinataires des données' %}

{% blocktrans %} Ces données sont conservées tant que le site Dezède existe. Les comptes contributeurs ne peuvent pas être effacés. Leur conservation est nécessaire afin de maintenir le lien entre un contributeur et ses publications. {% endblocktrans %}

{% blocktrans %} L’identité, la biographie renseignée par le contributeur et les publications sont publiés sur le site internet Dezède et accessibles à tous. {% endblocktrans %}

{% trans 'Droits des contributeurs sur leurs données' %}

{% blocktrans %} Les personnes détentrices d’un compte contributeur peuvent accéder et rectifier les données les concernant via leur espace contributeur. {% endblocktrans %}

{% blocktrans %} Les contributeurs ont également la possibilité de demander l’effacement des données les concernant, la limitation du traitement des données, de s’opposer au traitement et de définir des directives relatives au sort des données après leur mort. {% endblocktrans %}

{% blocktrans %} Ces droits ne sont pas absolus et les contributeurs les exerçant pourront se voir opposer un refus dument justifié. Les demandes d’exercice des droits doivent être transmises à l’adresse dezede.contact@gmail.com. {% endblocktrans %}

{% blocktrans %} Les publications postées sur le site internet Dezède sont susceptibles de contenir des données à caractère personnel relatives aux interprètes, compositeurs, auteurs et à toute personne ayant participé à la programmation étudiée. {% endblocktrans %}

{% trans 'Finalité et base légale du traitement' %}

{% blocktrans %}

Les données à caractère personnel utilisées dans ce cadre ont pour objet de permettre de :

  • reconstituer des programmes de spectacles dans un espace géographique et temporel circonscrit, établir la programmation d’une institution, retracer la carrière d’un interprète, ou étudier la diffusion et la réception d’une œuvre ;
  • publier ces corpus documentaires en ligne sur le site.
{% endblocktrans %} {% blocktrans %}

À cette fin, les données suivantes peuvent être traitées, dans le respect du principe de minimisation des données posé par le Règlement général sur la protection des données :

  • Etat-civil : nom, prénom, lieu de naissance, date de naissance … ;
  • Vie professionnelle : œuvre interprétée, lieu d’interprétation, métier… ;
  • Toute donnée pertinente pour la reconstruction de chronologies des spectacles.
{% endblocktrans %}

{% blocktrans %} Ces données sont issues de diverses sources publiques : archives (départementales, d’opéras, de salles de concert, de bibliothèques, de médiathèques, …), articles de presse, brochures de festivals, sites internet, … {% endblocktrans %}

{% blocktrans %} Le traitement de ces données repose sur les missions d’intérêt public (recherche et formation) dont sont investis les propriétaires du site (universités et laboratoires). {% endblocktrans %}

{% trans 'Durée de conservation et destinataires des données' %}

{% blocktrans %} Les données sont conservées sans limite de temps, en application de l’article 80 – 1 de la loi informatique et libertés relatif aux traitements mis en œuvre aux fins d’expression universitaire. {% endblocktrans %}

{% blocktrans %} Les données sont publiées sur le site internet Dezède et accessibles à tous. {% endblocktrans %}

{% trans 'Droits des personnes concernées sur leurs données' %}

{% blocktrans %} Toute personne dont des données à caractère personnel figurent dans les publications du site dispose de droits sur ces données. {% endblocktrans %}

{% blocktrans %} Elle peut accéder aux données la concernant directement depuis le site internet Dezède. {% endblocktrans %}

{% blocktrans %} Elle dispose du droit de demander la rectification des données la concernant, leur effacement, la limitation du traitement des données, de s’opposer au traitement et de définir des directives relatives au sort des données après leur mort. {% endblocktrans %}

{% blocktrans %} Ces données étant utilisées à des fins de recherche et d’expression universitaire, l’exercice de ces droits est soumis à conditions et pourra faire l’objet d’un refus dûment justifié. Les demandes d’exercice des droits doivent être transmises à l’adresse dezede.contact@gmail.com. {% endblocktrans %}
